# Backend - Leave Management System (MongoDB)
## Setup
1. Install dependencies:
   ```
   cd backend
   npm install
   ```
2. Copy .env.example to .env and set values.
3. Seed a manager and example users (optional):
   ```
   npm run seed
   ```
4. Run server:
   ```
   npm run dev
   ```
## Notes
- Server runs on PORT from .env (default 5000)
- API base: http://localhost:5000/api
