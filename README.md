# Leave Management System (MongoDB) - Ready Project
This zip contains two folders: `backend/` and `frontend/`.
## Quick start
1. Unzip `leave-management.zip`
2. Backend:
   - cd backend
   - cp .env.example .env
   - npm install
   - npm run seed   # optional: creates manager@example.com manager123
   - npm run dev
3. Frontend:
   - cd frontend
   - cp .env.example .env
   - npm install
   - npm run dev
4. Open frontend URL printed by Vite (usually http://localhost:5173) and login using seeded credentials.
