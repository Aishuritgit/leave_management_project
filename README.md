# Leave Management System (MongoDB) -  Project
This contains two folders: `backend/` and `frontend/`.
## Quick start
1. Backend:
   - cd backend
   - cp .env.example .env
   - npm install
   - npm run seed   # optional: creates manager@example.com manager123
   - npm run dev
1. Frontend:
   - cd frontend
   - cp .env.example .env
   - npm install
   - npm run dev
3. Open frontend URL printed by Vite (usually http://localhost:5173) and login using seeded credentials.
